# Проектирование приложения для доставки еды

## Описание системы

Приложение для доставки еды позволяет пользователям заказывать еду из различных ресторанов и получать её с помощью курьеров. Основные участники системы включают клиентов (пользователей), ресторан, службу поддержки и курьеров. Не знаю, какое дополнительное пояснение нужно написать, потому что никаких особенностей в ней нет
