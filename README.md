# Проектирование приложения для доставки еды

## Описание системы

Приложение для доставки еды позволяет пользователям заказывать еду из различных ресторанов и получать её с помощью курьеров. Основные участники системы включают клиентов (пользователей), рестораны и курьеров. Клиенты могут просматривать меню, оформлять заказы, отслеживать их статус и оплачивать заказы через приложение. Рестораны могут управлять своими меню, обрабатывать заказы и взаимодействовать с курьерами для организации доставки. Курьеры принимают заказы на доставку, получают информацию о маршрутах и доставляют еду клиентам. Система должна обеспечивать взаимодействие между всеми участниками, обеспечивая точность и своевременность выполнения заказов.
